Fight all knights

Includes Design, KoL mafia

About

Not A KoL dev. Script as follows.
Script requires IoTM: can of Rain-dough (black box), Source Terminal (Digitize), Witchess Set.
Skill required: Frigidalmation. Otherwise, edit the script or custom combat scripts so you kill the knights.
Finally: Check your piece id number. Mine is 1936. You can check this in the KoL mafia relay in your browser, just highlight the piece you wish to fight. The browser should display the following
"choice.php?option=1&pwd=(YOURPASSWORDHASH!)&whichchoice=1182&piece=(PIECEID)".
Those last digits are what you want. Change pieceid in the script (line 2) to the number you want to target.

Functionality:

Automatically beats down on the Witchess knights, digitizing the first, then uses all black box clones, and uses up all remaining witchess fights.
